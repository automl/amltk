    Path loaders

    Loaders for PathBuckets.

    The Loaders in this module are used to load and save objects identified by a unique Path. For saving objects, these loaders rely on checking the type of the object for can_save and save methods. For loading objects, these loaders rely on checking the file extension of the path for can_load and load methods.

    class PDLoader #

    Bases: PathLoader[DataFrame | Series]

    A Loader for loading and saving pd.DataFrames.

    This loader supports the following file extensions:

    • ".csv"
    • ".parquet"
    • ".pdpickle"

    This loader supports the following types:

    • pd.DataFrame
    • pd.Series - Only to ".pdpickle" files
    Multiindex support

    There is currently no multi-index support as we explicitly use index_col=0 when loading a ".csv" file. This is because we assume that the first column is the index to prevent Unamed columns from being created.

    Series support

    There is currently limited support for pandas series as once written to csv/parquet, they are converted to a dataframe with a single column. See this issue

    Please consider using ".pdpickle" instead.

    @override

    """::: amltk.store.paths.path_loaders.PathLoader.save"""  # noqa: D415
    # Most pandas methods only seem to support dataframes
